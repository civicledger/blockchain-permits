# Blockchain Permits
Civic Ledger have defined a standard for permits and licences issued via blockchains. 

## Terminology
Grant
Transfer
Redeem
Revoke
Reclaim

## Permit types
### Basic
```
Name
Type
Issuer
Terms URL (Must be SSL)
Icon URL
Start Time
End Time
Expiration (Date Time)
Transferable (Boolean)
Version
```
